Events List FAQs

How can I change the timeframe of displayed Events?

All Events are listed by default. To change the Timeframe covered, select either:

  • 'Today'
  • 'In 30 Days'
  • 'In 60 Days'
  • 'In 90 Days'

Can I expand the Events List to full screen?

Yes, at the bottom right of the report you will see an expand icon (a diagonal double-ended arrow). This button will enable you to view the listings in full screen mode.

Why are there no Events listed when I change the timeframe?

If no Events are listed, then there are no Events during the timeframe you've selected. If you believe this is the result of a technical issue, feel free to contact events@danrivermoms.com.

I've submitted an Event, but I do not see it?

All Events must be reviewed and approved per our Site Posting Guidelines. There also may be a small amount of lag time from approval before the data is available on this site. If it's been more than 24 hours since you've submitted an Event and you'd like more info, please contact events@danrivermoms.com.

I see the Event I've posted, but it is either wrong or needs to be changed. How can I do that?

For all corrections/changes, please email events@danrivermoms.com with the event name and specifics on desired changes. The email should come from the Event POC, and all changes must meet our Event Posting Guidelines.
